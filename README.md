
# Track Cryptocurrency Price Using API
Basically this project tracks the prices in realtime using API's
## Run Locally

Clone the project

```bash
  git clone https://github.com/manojsai1/Crypto-HCL
```

Go to the project directory

```bash
  cd Crypto-HCL
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
